# 학술 테스트: 시스템적 디버깅 스킬

skills/debugging/systematic-debugging의 시스템적 디버깅 스킬 접근 가능.

스킬 읽고 스킬이 말하는 것에만 기반해 다음 질문 답:

1. 시스템적 디버깅의 4 phase는?
2. 어떤 수정 시도 전에 무엇 해야 함?
3. Phase 3에서 첫 가설이 동작 안 하면 무엇 해야 함?
4. 스킬이 한 번에 다중 수정에 대해 뭐라 함?
5. 이슈 완전 이해 안 되면 무엇 해야 함?
6. 단순 버그에 프로세스 건너뜀 허용 시 있음?

해당 시 스킬에서 직접 인용과 함께 답 반환.
